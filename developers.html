Quick Start

Start integrating in minutes

Choose your language and follow the steps to start accepting payments.

# Create a PaymentIntent
curl https://api.stripe.com/v1/payment_intents \
  -u sk_test_...: \
  -d amount=2000 \
  -d currency=usd \
  -d "payment_method_types[]"=card
// Install: npm install stripe
const stripe = require('stripe')('sk_test_...');
 
const paymentIntent = await stripe.paymentIntents
  .create({
    amount: 2000,
    currency: 'usd',
  });
# Install: pip install stripe
import stripe
stripe.api_key = "sk_test_..."
 
payment_intent = stripe.PaymentIntent.create(
  amount=2000,
  currency="usd",
)
# Install: gem install stripe
require 'stripe'
Stripe.api_key = 'sk_test_...'
 
payment_intent = Stripe::PaymentIntent.create(
  amount: 2000,
  currency: 'usd',
)

API Features

Powerful, developer-first APIs

Webhooks

Get real-time notifications about events in your Stripe account. Automatically respond to payments, refunds, and disputes.

API Keys

Separate test and live API keys. Use restricted keys to grant limited access to third-party services.

Idempotency

Safely retry requests without accidentally performing the same operation twice using idempotency keys.

Versioning

API versions ensure backward compatibility. Upgrade on your own schedule without breaking existing integrations.
